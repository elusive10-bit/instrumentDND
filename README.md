# Instrument Drag and Drop

A responsive drag-and-drop musical instrument web app with touch support.  
Built with HTML, CSS, and JavaScript.

## Features
- Responsive layout for all devices
- Supports both mouse and touch drag-and-drop
- Lightweight and ready for GitHub Pages

## Deploying to GitHub Pages
1. Create a new public repository named `instrumentDragAndDrop` on GitHub.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Under "Source", select **Deploy from branch → main branch**.
5. Save — your app will be live at:

```
https://<your-username>.github.io/instrumentDragAndDrop/
```
